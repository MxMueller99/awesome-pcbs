## 💻 Microcontroller Boards
- [nRF54L15 miniDevKit](https://github.com/vksesham/nRF54L15_miniDevKit_Altium_PCB) - nRF54L15 SOC based PCB in Altium.
- [STMmicrocontrollerPCBdesign](https://github.com/harshaavenkatasalapathi-eng/STMmicrocontrollerPCBdesign) - 2 layer STM32 development board.
- [STM32F405RGT6 Dev Board](https://github.com/lendzeg/stm32f405-devboard) - Minimal STM32F4-based development board.
- [HelloBoardPCB](https://github.com/SalehGhobari/HelloBoardPCB/tree/main) - STM32F1-based development board.
- [PedroBoard](https://github.com/almtzr/PedroBoard) - Development board featuring multiple different MCUs.

## 🔬 Data Acquisition and Measurement 🔍
- [Crimpdeq PCB](https://github.com/crimpdeq/crimpdeq-pcb) - ESP based portable force sensor to measure finger strength.
- [MCP2515 CAN Bus Module](https://github.com/yasir-shahzad/MCP2515-CAN-Bus-Module) - MCP2515 and TJA1050 CAN bus module.

## 💾 Retro Projects
- [M8SBC-486](https://github.com/maniekx86/M8SBC-486) - 486 Homebrew computer project.
- [Kharon 386](https://github.com/UzixLS/pc-kharon-386) - IBM PC compatible machine with 386 CPU.
- [A4092](https://github.com/A4091/A4092) - Modern re-implementation of the A4091 SCSI controller.
- [OpenBrickGB](https://github.com/greigs/OpenBrickGB) - Lego based Game Boy project.
- [Latitude 5491 SSD Expander](https://github.com/a-little-wifi/Latitude-5491-SSD-Expander) - Dell Latitude 5491 SSD expander PCB.
- [SMD 3128 BM 3](https://github.com/Stargazer-24/smd-3128-bm-3) - SEGA Genesis/MegaDrive/32X reproduction cartridge.

## Miscellaneous
- [Fanpico](https://github.com/tjko/fanpico) - Raspberry Pico based programmable PWM fan controller. 
- [OpenPixel Risc-V Mouse PCB](https://github.com/MilitisTemplarii/OpenPixel-Risc-V-Mouse-PCB) - CH32V305 based high-end gaming mouse PCB.
- [RP2040 SmartWatch](https://github.com/MiscellaneousEngineer/RP2040-SM-PCB) - RP2040 based smartwatch.
- [DuPAL board](https://github.com/DuPAL-PAL-DUmper/DuPAL_Board/tree/master) - PCB to analyze PLDs and other memories.
- [Oro Link](https://github.com/oro-os/link/tree/master) - Set of custom PCBs for testing the Oro OS.
- [USB to UART Converter](https://github.com/adaptableCoder/USB-to-UART-CH340g-PCB) - Compact USB to UART CH340G based adapter.
- [LED Driver 3D](https://github.com/brainsmoke/leddriver3d) - ESP32 based LED driver for 3D sculptures.
- [Greyhound breakout PCB](https://github.com/mole99/greyhound-breakout-pcb) - Breakout board for the tinytapeout greyhound ASIC.
- [ChatGPT mod for Casio calculators](https://github.com/JonasHeselschwerdt/ChatGPT-mod-for-casio-calculators/tree/main) - ESP32S3 based ChatGPT modification for Casio calculators.
