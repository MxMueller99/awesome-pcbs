## 💻 Microcontroller Boards
- [nRF54L15 miniDevKit](https://github.com/vksesham/nRF54L15_miniDevKit_Altium_PCB) - nRF54L15 SOC based PCB in Altium.
- [STMmicrocontrollerPCBdesign](https://github.com/harshaavenkatasalapathi-eng/STMmicrocontrollerPCBdesign) - 2 layer STM32 development board.
- [STM32F405RGT6 Dev Board](https://github.com/lendzeg/stm32f405-devboard) - Minimal STM32F4-based development board.
- [HelloBoardPCB](https://github.com/SalehGhobari/HelloBoardPCB/tree/main) - STM32F1-based development board.
- [PedroBoard](https://github.com/almtzr/PedroBoard) - Development board featuring multiple different MCUs.
- [Mixed Signal STM32 Dev Board](https://github.com/JacobParent7/Mixed-Signal-STM32-Dev-Board) - STM32 based mixed signal PCB.
- [STM32F4xx Mixed Signal](https://github.com/amrithHN/stm32f4xx-mixed-signal-mutlilayer-pcb) - STM32F407vg based mixed signal Dev board.
- [Mixed Signal Controller](https://github.com/Abdelrhman20khaled/Embedded-Mixed-Signal-Controller-for-Audio-and-Motor-Systems) - STM32F407 based mixed signal board with Ethernet and a 24-bit DAC/ADC.

## Contributing
If you would like to add a project, open a pull request.

Please try to follow these rules:
- Only add repositories that include actual PCB design files, schematics, or layout files.
- Prefer projects with a decent documentation.
- Add a short one-line description.
- Avoid duplicates.
